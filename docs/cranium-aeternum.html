How to Install

System Requirements
  • RAM: 4GB minimum, 8GB recommended
  • Java: Java 17 or newer
  • Storage: At least 2GB free space
Installation Steps
  1. Download the .mrpack file using the button above
  2. Open your Minecraft launcher (Prism, MultiMC, ATLauncher, etc.)
  3. Create a new instance by importing the .mrpack file
  4. Wait for the launcher to download all required files
  5. Launch the game and enjoy!
NOTE: DO NOT UPDATE THIS MODPACK BY PRESSING THE UPDATE ALL BUTTON. IF YOU UPDATE YOUR MODS YOU WILL HAVE ISSUES CONNECTING TO THE SERVER.
